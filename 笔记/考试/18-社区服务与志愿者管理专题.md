# 社区服务与志愿者管理专题

## 一、社区服务体系

### 1. 社区服务概念
社区服务是指在党和政府领导下，由社区组织或服务组织发动社区居民开展的，旨在满足社区居民日常生活需求、提高社区居民生活质量的活动和服务。

### 2. 社区服务特点
- **福利性**：以社会福利为主，不以盈利为目的
- **区域性**：面向本社区居民
- **互助性**：居民自助与互助相结合
- **多样性**：服务内容涵盖方方面面
- **专业性**：逐步引入社会工作专业方法

### 3. 社区服务体系构成

```
社区服务体系
├── 公共服务（政府提供）
│   ├── 就业服务
│   ├── 社会保险
│   ├── 社会救助
│   └── 医疗卫生
├── 志愿服务（居民参与）
│   ├── 邻里互助
│   ├── 社区巡逻
│   └── 特殊帮扶
├── 商业服务（市场运作）
│   ├── 便利店
│   ├── 餐饮
│   └── 维修
└── 专业服务（社工机构）
    ├── 个案工作
    ├── 小组工作
    └── 社区工作
```

### 4. 社区服务主要领域

| 服务领域 | 具体内容 | 服务对象 |
|---------|---------|---------|
| 老年人服务 | 日间照料、居家养老、健康管理 | 60岁以上老人 |
| 儿童青少年服务 | 四点半课堂、兴趣培养、心理辅导 | 未成年人 |
| 残疾人服务 | 康复训练、就业指导、社会融入 | 残疾人 |
| 就业帮扶 | 技能培训、岗位推荐、创业支持 | 失业人员 |
| 心理健康 | 情绪疏导、危机干预、心理咨询 | 社区居民 |
| 矛盾调解 | 邻里纠纷、物业纠纷、家庭矛盾 | 纠纷当事人 |

## 二、志愿者管理

### 1. 志愿者概念
志愿者是指自愿贡献个人时间和精力，在不为物质报酬的前提下，为社会和他人提供服务的个人。

### 2. 志愿者特征
- **自愿性**：非强制，自主选择
- **无偿性**：不获取物质报酬
- **公益性**：服务于社会公共利益
- **业余性**：利用业余时间提供服务
- **组织性**：在一定组织框架内活动

### 3. 志愿者权利与义务

**志愿者权利：**
- 自愿加入或退出志愿服务组织
- 获得志愿服务活动真实、准确、完整的信息
- 获得志愿服务所需的条件和必要的保障
- 获得志愿服务知识和技能培训
- 对志愿服务组织的工作进行监督，提出意见和建议
- 请求解决问题和提出诉求
- 法律、法规规定的其他权利

**志愿者义务：**
- 遵守法律、法规，遵守志愿服务组织的章程
- 接受志愿服务活动所需的知识和技能培训
- 履行志愿服务承诺，完成志愿服务任务
- 服从志愿服务组织的安排和管理
- 尊重志愿服务对象的意愿，不得损害其合法权益
- 保守在志愿服务中接触到的个人隐私和商业秘密
- 法律、法规规定的其他义务

### 4. 志愿者管理流程

```
志愿者管理六步法
① 招募→② 选拔→③ 培训→④ 安排→⑤ 督导→⑥ 激励
```

#### 第一步：招募
- **渠道**：社区公告、微信公众号、志愿服务平台、学校合作、企业合作
- **方式**：自愿报名+组织动员相结合
- **注意事项**：明确志愿服务内容、时间、要求

#### 第二步：选拔
- **方式**：面谈、笔试、技能测试
- **内容**：动机评估、能力评估、素质评估
- **原则**：人岗匹配、择优录用

#### 第三步：培训
- **岗前培训**：志愿服务理念、基础知识、基本技能、安全教育
- **专业培训**：专项技能、服务方法、应急处理
- **在岗培训**：持续学习、能力提升

**培训内容框架：**
```
必修模块：
├─ 志愿服务概论（2学时）
├─ 志愿服务伦理与原则（2学时）
├─ 志愿服务安全与自我保护（1学时）
└─ 社区概况与服务对象认知（1学时）

选修模块：
├─ 老年人服务技巧
├─ 儿童青少年服务技巧
├─ 残疾人服务技巧
├─ 应急救护技能
└─ 沟通与冲突处理技巧
```

#### 第四步：安排
- **岗位匹配**：根据志愿者特长、时间、意愿进行合理分配
- **工作分配**：明确任务内容、工作标准、完成时限
- **过程监控**：建立联络机制，定期跟踪了解情况

#### 第五步：督导
- **目的**：保证服务质量、支持志愿者成长、解决问题困难
- **方式**：
  - 现场督导：实地巡查、现场指导
  - 会议督导：定期开会、集体研讨
  - 个别督导：一对一指导、情绪支持
  - 同伴督导：志愿者相互支持与学习
- **督导内容**：
  - 服务质量监控
  - 问题与困难协助解决
  - 志愿者情绪支持
  - 关系协调与资源链接

#### 第六步：激励
- **精神激励**：表彰证书、荣誉墙、先进评选、媒体报道
- **物质激励**：交通补贴、餐饮补贴、保险保障、纪念品
- **发展激励**：培训机会、晋升机会、推荐信、社会实践证明
- **情感激励**：感谢信、生日祝福、节日问候、团队活动

### 5. 志愿服务时长认定与记录
- 每次服务后如实记录服务时间、内容、成效
- 采用"志愿四川"等志愿服务信息平台进行电子记录
- 服务满一定时长可颁发志愿服务证书
- 星级志愿者认定：100h（一星）、300h（二星）、600h（三星）、1000h（四星）、1500h（五星）

## 三、社区社会组织

### 1. 社区社会组织概念
社区社会组织是指由社区居民自发成立，在社区范围内开展活动，满足社区居民多样化需求的社会组织。

### 2. 社区社会组织类型

| 类型 | 示例 | 主要功能 |
|-----|------|---------|
| 维权类 | 业主委员会、居民议事会 | 利益表达、民主参与 |
| 服务类 | 养老服务队、帮扶小组 | 便民服务、邻里互助 |
| 文化类 | 文艺队、书画协会、运动队 | 文化娱乐、精神需求 |
| 公益类 | 志愿服务队、环保队 | 公益事业、社区治理 |
| 兴趣爱好类 | 棋牌社、摄影协会 | 丰富生活、增进交流 |

### 3. 社区社会组织培育
- **备案管理**：社区社会组织到街道备案即可开展活动
- **资金支持**：政府购买服务、社区公益金支持
- **场地支持**：提供活动室、多功能厅等场所
- **能力建设**：组织管理培训、专业督导支持
- **资源链接**：帮助对接企业、社会组织资源

## 四、"时间银行"互助养老

### 1. 概念
"时间银行"是指志愿者为老年人提供志愿服务，服务时长存入"时间银行"，待自己年老需要帮助时，可提取存储的时间换取他人志愿服务。

### 2. 运行机制
- 志愿者提供服务 → 记录服务时长 → 存入时间银行
- 需要帮助时 → 申请支取时间 → 获得对应时长的志愿服务

### 3. 注意事项
- 服务时长不等价兑换货币
- 服务记录需真实、准确、完整
- 建立完善的监督和管理制度
- 注意保护服务对象隐私和安全

## 五、社区特殊群体服务

### 1. 老年人服务
- **日间照料中心**：为白天无人照料的老人提供生活照料、康复训练、文化娱乐
- **居家养老服务**：助餐、助洁、助浴、助医、助行、代办
- **特殊老人帮扶**：独居老人、失能老人、失独老人、留守老人
- **老年人防骗**：常见保健品诈骗、电信诈骗、理财诈骗防范宣传

### 2. 儿童青少年服务
- **四点半课堂**：放学后作业辅导、兴趣培养
- **暑期托管班**：暑期照看、技能培训
- **困境儿童帮扶**：事实无人抚养儿童、留守儿童关爱保护
- **心理健康**：关注青少年心理问题，开展心理疏导

### 3. 残疾人服务
- **康复服务**：社区康复、家庭康复
- **就业服务**：技能培训、岗位推荐、辅助性就业
- **无障碍改造**：坡道、扶手、低位灶台等
- **社会融入**：文体活动、志愿者结对、社会参与

## 六、考点点睛与真题练习

### 核心考点速记

```
【考点1】社区服务特点：福利性、区域性、互助性、多样性、专业性
【考点2】志愿者三大特征：自愿、无偿、公益
【考点3】志愿者管理六步法：招募→选拔→培训→安排→督导→激励
【考点4】志愿服务时长认定：100/300/600/1000/1500小时=五个星级
【考点5】社区社会组织三类：维权类、服务类、文化/公益类
【考点6】时间银行：服务时长存储，老年时支取，不兑换货币
【考点7】四社联动：社区+社会组织+社会工作者+志愿者
```

1. 社区服务的核心特征是（ ）

   C. 时间银行只面向老年人开放
   D. 志愿者服务时长可以继承给子女

**简答题：**

9. 简述志愿者管理的基本流程。
   **参考答案：**
   志愿者管理的基本流程包括六个环节：
   ① 招募：通过多种渠道发布信息，吸引有意愿者报名；
   ② 选拔：通过面谈、测试等方式筛选合适人员；
   ③ 培训：开展岗前培训和专项培训，确保服务质量；
   ④ 安排：根据志愿者特长进行岗位匹配和工作分配；
   ⑤ 督导：监督指导志愿服务过程，解决实际问题；
   ⑥ 激励：对优秀志愿者进行表彰奖励，保持积极性。
