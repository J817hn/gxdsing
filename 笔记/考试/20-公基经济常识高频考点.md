# 公基经济常识高频考点

## 一、商品与货币

### 1. 商品的基本属性
- **使用价值**：商品能满足人们某种需要的属性（有用性）
- **价值**：凝结在商品中无差别的人类劳动
- **二者关系**：使用价值是价值的物质承担者，价值是交换价值的基础

### 2. 货币的本质与职能

**货币的本质：** 一般等价物

**货币的职能：**
| 职能 | 含义 | 举例 |
|-----|------|-----|
| 价值尺度 | 衡量商品价值大小 | 标价 |
| 流通手段 | 充当商品交换媒介 | 买卖行为 |
| 支付手段 | 清偿债务、支付工资等 | 还债、工资 |
| 贮藏手段 | 退出流通储存起来 | 存款 |
| 世界货币 | 在世界市场上发挥作用 | 国际贸易 |

**流通规律：** 流通中需要的货币量 = 待售商品价格总额 ÷ 货币流通速度

### 3. 纸币与通货膨胀
- **纸币发行量**：超过实际需要的货币量 → 通货膨胀（物价上涨，货币贬值）
- **纸币发行量**：少于实际需要 → 通货紧缩（物价下跌，经济衰退）
- **通货膨胀成因**：货币发行过多、需求过剩、成本上升、结构性因素

---

## 二、价值规律

### 1. 价值规律内容
- 商品的价值量由生产商品的社会必要劳动时间决定
- 商品交换以价值量为基础，实行等价交换

### 2. 价值规律表现形式
> **价格围绕价值上下波动**（供求关系影响价格）

```
价值规律图示：
价格 ↗ 供不应求 → 价格上涨 → 生产扩大 → 供过于求
价格 ↘ 供过于求 → 价格下跌 → 生产缩小 → 供不应求
```

### 3. 价值规律作用
- 调节生产资料和劳动力在各生产部门的分配
- 刺激商品生产者改进技术、改善管理、提高劳动生产率
- 促使商品生产者在竞争中优胜劣汰

## 三、社会主义市场经济

### 1. 基本经济制度
**公有制为主体、多种所有制经济共同发展**

| 所有制 | 地位 | 形式 |
|-------|------|-----|
| 全民所有制（国有经济） | 主导力量 | 国有企业 |
| 集体所有制（集体经济） | 重要组成部分 | 集体企业、合作社 |
| 个体经济 | 社会主义市场经济补充 | 个体工商户 |
| 私营经济 | 社会主义市场经济补充 | 私营企业 |
| 外商经济 | 社会主义市场经济补充 | 中外合资、外商独资 |

**公有制主体地位体现：**
- 公有资产在社会总资产中占优势
- 国有经济控制国民经济命脉
- 主导作用体现在控制力上

### 2. 社会主义市场经济特点
- **所有制结构**：公有制为主体，多种所有制共同发展
- **分配制度**：按劳分配为主体，多种分配方式并存
- **运行机制**：计划和市场两种手段相结合
- **根本目标**：共同富裕
- **、微观调控**：国家宏观调控能力强

### 3. 资源配置方式
| 方式 | 市场调节 | 计划（政府）调节 |
|-----|---------|----------------|
| 优点 | 灵敏、高效、自动 | 宏观总量平衡、长远规划 |
| 缺点 | 自发性、盲目性、滞后性 | 信息不完全、效率损失 |
| 适用 | 竞争性行业、微观领域 | 公共物品、自然垄断 |

### 4. 社会主义市场经济体制框架
```
社会主义市场经济体制
├── 现代企业制度（微观基础）
├── 市场体系（资源配置场所）
├── 宏观调控体系（政府作用）
├── 收入分配制度（激励机制）
└── 社会保障制度（稳定器）
```

## 四、宏观经济指标与调控

### 1. 国民经济核算指标

**GDP（国内生产总值）**
- 地域概念：在一国领土范围内生产的最终产品和服务
- 核算方法：生产法、收入法、支出法
- **注意**：不包含非生产性交易（股票买卖、二手房交易）、非法交易

**GNP（国民生产总值）**
- 国民概念：一国居民（含企业和个人）在他国和本国生产的最终产品和服务

**GDP与GNP关系：**
> GDP = GNP - 本国人在外国生产的产值 + 外国人在本国生产的产值

### 2. 经济增长与经济发展

**经济增长：** GDP总量增加（量的概念）

**经济发展：** 不仅包括经济增长，还包括经济结构优化、社会进步、民生改善（质的概念）

**经济增长方式：**
- **粗放型**：主要靠增加投入、扩大规模实现增长
- **集约型**：主要靠提高效率、技术进步实现增长

### 3. 宏观调控目标与手段

**宏观调控目标：**
- 促进经济增长
- 增加就业
- 稳定物价
- 保持国际收支平衡

**宏观调控手段：**

| 手段 | 内容 | 特点 |
|-----|------|-----|
| 经济手段 | 财政政策、货币政策 | 间接调控，市场化 |
| 法律手段 | 立法、司法、执法 | 规范性、强制性 |
| 行政手段 | 行政命令、指示、规定 | 直接、迅速，但慎用 |

**财政政策：**
- **扩张性财政政策**（经济萧条时采用）：减税、增加支出
- **紧缩性财政政策**（经济过热时采用）：增税、减少支出
- **工具**：税、债、支出

**货币政策：**
- **扩张性货币政策**（经济萧条时采用）：降低利率、降低存款准备金率、公开市场买入
- **紧缩性货币政策**（经济过热时采用）：提高利率、提高存款准备金率、公开市场卖出
- **工具**：利率、存款准备金率、公开市场业务、再贴现率

## 五、财政收入与支出

### 1. 财政收入形式
- 税收（最主要形式，占90%以上）
- 国债
- 国有资产收益
- 专项收入（教育费附加等）
- 其他收入

### 2. 财政支出分类
| 按经济性质 | 按功能 |
|-----------|-------|
| 经济建设支出 | 经济建设 |
| 社会文教支出 | 社会文教 |
| 行政管理支出 | 行政管理 |
| 国防支出 | 国防 |
| 债务支出 | 债务付息 |
| 其他支出 | 社会保障、补贴等 |

### 3. 财政收支关系
- **财政盈余**：收入 > 支出
- **财政赤字**：收入 < 支出
- **财政平衡**：收入 ≈ 支出（略有盈余或赤字）

## 六、个人收入分配

### 1. 分配制度
**按劳分配为主体、多种分配方式并存**

### 2. 按劳分配
- 在公有制经济中
- 以劳动贡献为依据分配
- 体现了劳动的平等

### 3. 按生产要素分配
- **资本要素**：利息、股息、分红
- **土地要素**：地租、土地转让费
- **劳动要素**：工资、薪金
- **技术要素**：技术入股、专利转让费
- **管理要素**：管理绩效年薪

### 4. 效率与公平
- **效率**：资源配置的优化程度
- **公平**：收入分配中人们对等交换关系的主观感受
- **关系**：效率是公平的物质基础，公平是效率的制度保障
- **政策取向**：
  - 效率优先、兼顾公平（初次分配）
  - 更加注重公平（再分配）

## 七、微观经济主体

### 1. 企业
- **定义**：依法设立，独立从事生产经营活动，以盈利为目的的经济组织
- **组织形式**：个人独资企业、合伙企业、公司制企业
- **经营目标**：利润最大化
- **法律责任**：
  - 个人独资企业：无限连带责任
  - 合伙企业：无限连带责任（普通合伙人）
  - 有限责任公司：以其全部资产对债务承担责任
- **公司类型**：有限责任公司、股份有限公司

### 2. 现代企业制度特征
- 产权清晰
- 权责明确
- 政企分开
- 管理科学

### 3. 企业经营与发展
- **经营策略**：市场定位、产品策略、价格策略、促销策略
- **经营原则**：合法经营、诚实守信
- **企业形象**：品牌、信誉、社会责任
- **企业重组**：兼并、收购、合并、分立

---

## 八、市场体系与市场竞争

### 1. 市场体系
- **商品市场**：消费品市场、生产资料市场
- **要素市场**：金融市场、劳动力市场、土地市场、技术市场、信息市场
- **市场秩序**：公平竞争、诚实守信、法制保障

### 2. 市场竞争类型
| 类型 | 特征 | 举例 |
|-----|------|-----|
| 完全竞争 | 产品无差异，买卖双方都是价格接受者 | 农产品市场 |
| 垄断竞争 | 产品有差别，存在一定垄断 | 餐饮、零售 |
| 寡头垄断 | 少数企业控制市场 | 电信、石油 |
| 完全垄断 | 一家企业独占市场 | 电力、自来水 |

### 3. 市场失灵
- **原因**：外部性、公共物品、信息不对称、垄断
- **表现**：资源配置效率降低、社会公平受损
- **矫正**：政府干预、公共管制、建立健全市场规则
