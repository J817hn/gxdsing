## 一、社区基础概念

### 1. 社区的定义
- **社会学定义**：社区是若干社会群体或社会组织聚集在某一地域里形成的一个在生活上相互关联的大集体。
- **社区工作**：在党和政府的领导下，依靠社区力量，利用社区资源，强化社区功能，解决社区问题，促进社区政治、经济、文化、环境协调发展的社会工作。

### 2. 社区的构成要素（五要素）
| 要素 | 说明 |
|------|------|
| 人口 | 数量、质量、结构、分布 |
| 地域 | 自然地理条件、空间范围 |
| 设施 | 生产和生活设施 |
| 文化 | 价值观、行为规范、传统文化 |
| 组织 | 社区自治组织、社会组织 |

### 3. 社区的类型
| 分类标准 | 类型 |
|----------|------|
| 按空间特征 | 法定社区、自然社区、专能社区、虚拟社区 |
| 按地域 | 城市社区、农村社区、城镇社区 |
| 按功能 | 居住社区、商业社区、工业社区 |

## 二、居民委员会（居委会）【最高频】

### 1. 居委会的性质
**基层群众性自治组织**（"自我管理、自我教育、自我服务"）
> 注意：不设区的市、市辖区的人民政府或街道办事处对居委会工作给予**指导、支持和帮助**，而非领导关系。

### 2. 居委会的组成（2026年新法修订重点）
- 由主任、副主任和委员共**5-9人**组成
- 成员中**应当有妇女成员**
- 实行**近亲属回避**
- 居委会主任可由**社区党组织负责人**通过法定程序担任
- 居委会成员与社区党组织领导班子成员可以**交叉任职**

### 3. 居委会的下属委员会（5大委员会）
1. **人民调解委员会** — 调解民间纠纷
2. **治安保卫委员会** — 协助维护社会治安
3. **公共卫生委员会** — 办理公共卫生事务
4. **环境和物业管理委员会** — 环境和物业管理
5. **老年人和妇女儿童工作委员会** — 关爱老年人、妇女儿童

### 4. 居委会的9大职责【背熟】
1. 宣传宪法、法律、法规和国家政策，推动居民履行法定义务，维护居民合法权益
2. 引导居民践行**社会主义核心价值观**，铸牢**中华民族共同体意识**，倡导文明健康绿色环保生活方式
3. 办理公共事务和公益事业，开展便民利民社区服务，关心关爱困难居民
4. **调解民间纠纷**，促进家庭和睦、邻里和谐、社区安宁
5. 协助维护社会治安，组织群防群治，协助处理信访、矛盾纠纷、**社区矫正**和**刑满释放人员帮教**
6. 支持引导**社会组织、社会工作者、志愿者**参与社区治理
7. 指导和协助设立业主大会和选举业主委员会，调解物业纠纷
8. 协助人民政府或街道办事处做好与居民利益有关的工作
9. 向政府反映居民意见、要求和提出建议

### 5. 居委会的任期
- 每届任期**5年**，可以连选连任

### 6. 居委会的选举方式
- 由本社区全体有选举权的居民或每户派代表选举
- 也可由居民代表选举
- 由**居民选举委员会**主持

### 7. 居委会的法人资格
- 具有**基层群众性自治组织法人资格**
- 主任为法定代表人
- 可从事履行职能所需的民事活动，**但不得提供担保**

## 三、社区治理【高频】

### 1. 社区治理的核心原则
- **党建引领** — 坚持中国共产党的领导
- **共建共治共享** — 多元主体参与
- **三社联动** — 社区、社会组织、社会工作者协调运作

### 2. 社区治理的主要模式
| 模式 | 特点 |
|------|------|
| 政府主导型 | 政府推动，社区执行 |
| 社区自治型 | 居民自主管理 |
| 多元共治型 | 政府+社区+社会组织+居民 |
| 网格化管理 | 精细化、信息化管理 |

### 3. 网格化管理
- 将社区划分为若干网格，配备网格员
- 职责：信息采集、便民服务、矛盾化解、治安防控、流动人口和特殊人群服务管理

## 四、社区服务【高频考点】

### 1. 社区服务的本质特征
- **福利性**（本质特征）
- 地域性、综合性、群众性、互助性

### 2. 社区服务的类型
| 类型 | 内容 |
|------|------|
| 公共服务 | 政府提供的公共产品和服务 |
| 公益服务 | 社会组织和志愿者提供的无偿服务 |
| 便利服务 | 满足居民日常生活需求的有偿服务 |

### 3. 社区工作者角色
| 角色 | 说明 |
|------|------|
| 服务者 | 为社区居民提供专业服务 |
| 管理者 | 协调社区资源、管理社区事务 |
| 协调者 | 协调各方关系，化解矛盾 |
| 教育者 | 提升居民素质，倡导文明风尚 |
| 倡导者 | 反映居民需求，推动政策 |

## 六、易混辨析

| 区别点 | 居委会 | 村委会 |
|--------|--------|--------|
| 适用范围 | 城市社区 | 农村 |
| 设立范围 | 一般1000-3000户 | 由乡政府决定 |
| 主管法律 | 居委会组织法 | 村委会组织法 |
| 任务侧重点 | 城市管理、社会服务 | 农业生产、土地管理 |

## 七、经典真题回顾

**【单选】居委会是什么性质的組織？**
> 答案：基层群众性自治组织（自我管理、自我教育、自我服务）

**【多选】居委会的下属委员会包括？**
> 答案：人民调解、治安保卫、公共卫生、环境和物业管理、老年人和妇女儿童工作委员会

**【判断】居委会可以对外提供担保。**
> 答案：错误。不得提供担保。

**【单选】居委会每届任期多少年？**
> 答案：5年
