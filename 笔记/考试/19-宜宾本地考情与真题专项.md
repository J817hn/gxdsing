## 二、宜宾市情要点（必背）

### 2.1 基本市情
| 项目 | 内容 |
|-----|------|
| 位置 | 四川省东南部，长江上游 |
| 面积 | 约1.33万平方公里 |
| 人口 | 约550万（2023年） |
| 下辖 | 翠屏区、叙州区、南溪区 + 8个县 |
| 别称 | "万里长江第一城"、中国酒都 |
| GDP | 四川省第三（仅次于成都、绵阳） |
| 特色产业 | 动力电池、晶硅光伏、白酒、智能终端 |

### 2.2 宜宾市基层治理特色政策

**"全科网格"建设**
- 宜宾市推进"全科网格"建设，整合党建、综治、城管、应急等多网合一
- 网格员"一员多能"，承担信息采集、问题处置、便民服务等职能
- 2024年，宜宾市网格化服务管理覆盖率达到100%

**"红色物业"**
- 宜宾市推进社区物业党建联建，提升物业管理服务水平
- 建立社区党委—小区党支部—楼栋党小组三级组织体系
- 重点解决无物业管理老旧小区问题

**"两项改革"后半篇**
- 宜宾市做好乡镇行政区划和村级建制调整改革"后半篇"文章
- 优化资源配置，提升基层治理效能
- 做强中心镇、中心村，发挥辐射带动作用

**"心立方"群众工作法**
- "听"：深入群众，听民声、察民情
- "记"：记录群众诉求，建立台账
- "办"：分类处理，限时办结
- "评"：群众评价，闭环管理

### 3.4 简答题

**16. 简述宜宾市"心立方"群众工作法的主要内容。**

**参考答案：**
"心立方"群众工作法是宜宾市创新群众工作的重要方法，包含四个核心环节：
- **"听"**：深入群众，倾听民声，察民情、解民忧
- **"记"**：认真记录群众诉求，建立问题台账
- **"办"**：对群众诉求分类处理，明确责任，限时办结
- **"评"**：群众评价监督，形成闭环，提升满意度

**核心要义**：变"被动应对"为"主动服务"，以群众需求为导向，实现服务群众全覆盖。

---

**17. 简述宜宾市"红色物业"的主要做法。**

**参考答案：**
宜宾市"红色物业"是以党建引领提升物业管理服务水平的重要举措，主要做法包括：
1. **组织引领**：建立社区党委—小区党支部—楼栋党小组三级组织体系
2. **机制创新**：推行"双向进入、交叉任职"，吸纳物业企业负责人担任社区兼职委员
3. **服务提升**：将物业企业打造成党组织联系服务群众的工作队伍
4. **重点突破**：针对无物业管理的老旧小区，通过"红色物业"实现兜底管理
5. **共治共享**：整合社区、物业、业委会、居民等多方力量，共同参与社区治理
