## 二、经济常识

### 1. 市场经济基本概念
| 概念 | 说明 |
|------|------|
| GDP | 国内生产总值（地域口径） |
| GNP | 国民生产总值（国民口径） |
| 恩格尔系数 | 食品支出占消费支出比重，比重越高生活水平越低 |
| 基尼系数 | 衡量收入分配差距，0-1之间，越大越不平等 |

### 2. 社会主义市场经济体制
- 基本制度：公有制为主体、多种所有制经济共同发展
- 市场在资源配置中起**决定性作用**
- 更好发挥政府作用（宏观调控）

### 3. 财政与税收
- 财政的作用：促进社会公平、改善民生、国家治理的基础
- 主要税种：增值税、消费税、企业所得税、个人所得税

## 三、法律常识【高频】

### 1. 宪法（根本法）
| 公民基本权利 | 内容 |
|-------------|------|
| 平等权 | 法律面前一律平等 |
| 政治权利 | 选举权、被选举权 |
| 宗教信仰自由 | — |
| 人身自由 | 人格尊严不受侵犯 |
| 社会经济权利 | 劳动权、休息权 |
| 教育文化权利 | 受教育权 |

### 2. 民法典（高频）
| 原则 | 说明 |
|------|------|
| 平等原则 | 民事主体地位平等 |
| 自愿原则 | 意思自治 |
| 公平原则 | 权利义务对等 |
| 诚信原则 | 诚实守信 |
| 公序良俗 | 遵守公共秩序和善良风俗 |

**社区相关民法考点：**
- 相邻关系：不动产相邻各方应当按照有利生产、方便生活原则处理
- 物业服务合同：业主委员会与物业服务人订立的合同对全体业主有约束力
- 侵权责任：高空抛物致害，公安机关调查后难以确定具体侵权人的，可能加害建筑物使用人给予补偿

### 3. 行政法
- **行政行为**：行政主体行使行政职权做出的直接影响相对人权利义务的行为
- **行政复议**：向做出该行为的上一级机关或本级人民政府申请
- **行政诉讼**：向人民法院提起

### 4. 社区工作常用法律
| 法律 | 核心内容 |
|------|----------|
| 《城市居委会组织法》 | 2026年1月1日起施行修订版 |
| 《村委会组织法》 | 农村基层群众自治 |
| 《物业管理条例》 | 业主大会、业委会、物业服务 |
| 《信访工作条例》 | 信访人权利义务、程序 |
| 《人民调解法》 | 调解民间纠纷 |
