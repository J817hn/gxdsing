# 劳动法考试要点

> 本文档整理社区工作人员招聘考试中劳动法相关考点，涵盖核心法条、常见题型及记忆要点，助您高效备考。

---

## 一、基础知识框架

### 1.1 法律体系概览

劳动法律体系以《中华人民共和国劳动法》为核心，包括多部配套法律法规：

| 法律法规 | 施行时间 | 核心要点 |
|---------|---------|---------|
| 《中华人民共和国劳动法》 | 1995年1月1日 | 劳动基本法，确立制度框架 |
| 《中华人民共和国劳动合同法》 | 2008年1月1日 | 劳动合同订立、履行、变更、解除 |
| 《中华人民共和国劳动争议调解仲裁法》 | 2008年5月1日 | 劳动争议处理程序 |
| 《工伤保险条例》 | 2010年修订 | 工伤认定、待遇、赔偿 |
| 《职工带薪年休假条例》 | 2008年1月1日 | 年休假制度 |

### 1.2 适用范围

**适用对象**：中华人民共和国境内的企业、个体经济组织、民办非企业单位等组织（以下统称用人单位）与劳动者建立劳动关系，适用劳动法。

**特殊说明**：
- 国家机关、事业单位、社会团体和与其建立劳动关系的劳动者，依照劳动法执行
- 公务员和参照公务员管理的人员不适用劳动法
- 农村劳动者（乡镇企业职工和进城务工、经商的农民除外）、现役军人、家庭保姆等不适用劳动法

## 二、劳动合同核心考点

### 2.1 劳动合同的订立

**订立原则**：订立劳动合同，应当遵循合法、公平、平等自愿、协商一致、诚实信用的原则。

**必含条款**（法定条款）：
1. 用人单位的名称、住所和法定代表人或者主要负责人
2. 劳动者的姓名、住址和居民身份证或者其他有效身份证件号码
3. 劳动合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护和劳动条件

**可备条款**（约定条款）：
- 试用期
- 培训
- 保守秘密
- 补充保险
- 福利待遇

### 2.2 劳动合同期限

| 类型 | 期限规定 | 试用期限 |
|-----|---------|---------|
| 固定期限 | 约定终止时间 | 合同期≤1年：≤1个月；1年<合同期≤3年：≤2个月；合同期>3年/无固定期限：≤6个月 |
| 无固定期限 | 无约定终止时间 | 可约定或不可约定 |
| 以完成一定工作任务为期限 | 任务完成即终止 | 不得约定试用期 |

**【高频考点】试用期规定**：
- 试用期包含在劳动合同期限内
- 同一用人单位与同一劳动者只能约定一次试用期
- 试用期工资不得低于合同约定工资的80%，且不得低于当地最低工资标准
- 试用期内提前3日通知即可解除劳动合同（劳动者）

### 2.3 无固定期限劳动合同

**应当订立情形**：
1. 劳动者在该用人单位连续工作满十年的
2. 用人单位初次实行劳动合同制度或者国有企业改制重新订立劳动合同时，劳动者在该用人单位连续工作满十年且距法定退休年龄不足十年的
3. 连续订立二次固定期限劳动合同，且劳动者没有严重违反规章制度等情形，续订劳动合同的

**【记忆口诀】** 十周年+十年+二次续订 = 必须签订无固定期限

### 2.4 劳动合同无效

**无效情形**：
- 以欺诈、胁迫的手段或者乘人之危，使对方在违背真实意思的情况下订立
- 用人单位免除自己的法定责任、排除劳动者权利
- 违反法律、行政法规强制性规定

**效力认定**：
- 部分无效不影响其他部分效力的，其他部分仍然有效
- 劳动合同被确认无效，劳动者已付出劳动的，用人单位应当支付劳动报酬

## 三、工作时间与休息休假

### 3.1 标准工时制

| 项目 | 标准规定 |
|-----|---------|
| 工作时间 | 每日工作8小时，每周工作40小时 |
| 休息休假 | 每周至少休息1日 |
| 加班上限 | 每月不超过36小时 |

### 3.2 加班工资计算

| 加班类型 | 支付标准 |
|---------|---------|
| 工作日加班 | 不低于150% |
| 休息日加班（能安排补休的） | 不低于200% 或安排补休 |
| 法定节假日加班 | 不低于300% |

**【计算公式】**
```
日工资 = 月工资收入 ÷ 月计薪天数（21.75天）
小时工资 = 月工资收入 ÷（月计薪天数 × 8小时）

加班费 = 日工资 × 加班小时数 × 支付比例
```

### 3.3 法定节假日

全体公民放假的节日（共11天）：
- 元旦（1天：1月1日）
- 春节（3天：农历正月初一至初三）
- 清明节（1天：清明当日）
- 劳动节（1天：5月1日）
- 端午节（1天：端午节当日）
- 中秋节（1天：中秋当日）
- 国庆节（3天：10月1日至3日）

### 3.4 年休假

| 累计工作年限 | 年休假天数 |
|-------------|-----------|
| 满1年不满10年 | 5天 |
| 满10年不满20年 | 10天 |
| 满20年 | 15天 |

**【重要规定】**：
- 国家法定休假日、休息日不计入年休假假期
- 单位确因工作需要不能安排职工休年休假的，经职工本人同意，可以不安排，但须支付不低于日工资收入300%的未休年休假工资报酬

## 四、工资薪酬

### 4.1 工资分配原则

- 按劳分配为主体的多种分配方式并存
- 工资水平逐步提高
- 实行同工同酬

### 4.2 最低工资标准

**核心规定**：
- 劳动者在法定工作时间或劳动合同约定的工作时间内提供了正常劳动的前提下，用人单位应支付的最低劳动报酬
- 不包括：加班工资、夜班补贴、高温低温井下有毒有害等特殊工作环境津贴、伙食补贴、住房补贴等

**【考试要点】**：最低工资标准由省、自治区、直辖市人民政府规定，报国务院备案。

### 4.3 工资支付规定

| 项目 | 规定 |
|-----|-----|
| 支付形式 | 货币支付，按月支付，不得克扣或无故拖欠 |
| 支付日期 | 与劳动者约定，遇节假日提前至最近工作日 |
| 请假工资 | 按劳动合同约定支付 |

## 五、社会保险

### 5.1 社会保险种类

| 险种 | 缴费主体 | 考试重点 |
|-----|---------|---------|
| 养老保险 | 单位+个人 | 多缴多得、长缴多得 |
| 医疗保险 | 单位+个人 | 定点就医、医保目录 |
| 失业保险 | 单位+个人 | 非因本人意愿中断就业 |
| 工伤保险 | 单位缴纳，个人不缴 | 单位承担，无过错责任 |
| 生育保险 | 单位缴纳，个人不缴 | 与医疗保险合并实施 |

### 5.2 工伤认定

**应当认定工伤情形**：
1. 在工作时间和工作场所内，因工作原因受到事故伤害的
2. 工作时间前后在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害的
3. 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的
4. 患职业病的
5. 因工外出期间，由于工作原因受到伤害或者发生事故下落不明的
6. 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的

**视同工伤情形**：
1. 在工作时间和工作岗位，突发疾病死亡或者在48小时内经抢救无效死亡的
2. 在抢险救灾等维护国家利益、公共利益活动中受到伤害的
3. 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发的

**【重要时间点】**：
- 用人单位须在事故伤害发生之日起30日内提出工伤认定申请
- 劳动者本人或直系亲属可在1年内提出
- 工伤认定作出之日起60日内可申请行政复议，对复议决定不服可提起行政诉讼

### 5.3 工伤保险待遇

| 情形 | 待遇标准 |
|-----|---------|
| 停工留薪期 | 原工资福利待遇不变，不超过12个月（可延长至24个月） |
| 一次性伤残补助金 | 根据伤残等级1-10级，对应27个月至7个月本人工资 |
| 一次性工亡补助金 | 上一年度全国城镇居民人均可支配收入的20倍 |
| 丧葬补助金 | 6个月统筹地区上年度职工月平均工资 |

## 六、劳动保护

### 6.1 女职工特殊保护

| 保护内容 | 具体规定 |
|---------|---------|
| 禁忌从事 | 矿山井下、第四级体力劳动强度的劳动、其他禁忌从事的劳动 |
| 孕期保护 | 不得安排怀孕女职工延长工作时间或夜班劳动；怀孕7个月以上不得安排加班和从事夜班 |
| 产假 | 98天（含产前15天）；难产增加15天；多胞胎每多1个婴儿增加15天 |
| 哺乳期 | 不得安排哺乳未满1周岁婴儿的女职工延长工作时间或夜班劳动 |
| 流产假 | 怀孕未满4个月流产：15天；满4个月流产：42天 |

### 6.2 未成年工特殊保护

- 不得招用未满16周岁的未成年人（文艺、体育和特种工艺单位除外，须经审批）
- 不得安排未成年工从事矿山井下、有毒有害、国家规定的第四级体力劳动强度的劳动
- 应对未成年工定期进行健康检查

## 七、劳动争议处理

### 7.1 争议类型

| 争议类型 | 含义 |
|---------|-----|
| 劳动争议 | 用人单位与劳动者因劳动权利义务发生的纠纷 |
| 集体劳动争议 | 劳动者一方为10人以上，有共同请求的劳动争议 |

### 7.2 处理程序

```
┌──────────────────────────────────────────────────────────┐
│                    劳动争议处理流程                       │
├──────────────────────────────────────────────────────────┤
│                                                          │
│   ①协商和解 ←→ ②调解（企业劳动争议调解委员会/             │
│       ↑          基层人民调解组织/街道/乡镇调解组织）      │
│       ↓                              ↓                   │
│       └──── ③仲裁（劳动争议仲裁委员会）──────┐            │
│                                          ↓               │
│                              ④诉讼（人民法院）←─────────┘
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 7.3 仲裁时效与期限

| 项目 | 规定 |
|-----|-----|
| 仲裁申请时效 | 知道或应当知道权利被侵害之日起1年内 |
| 仲裁受理 | 收到仲裁申请之日起5日内决定是否受理 |
| 仲裁审理 | 45日内结束，复杂案件可延期15日 |
| 先予执行 | 追索劳动报酬、工伤医疗费等可裁定先予执行 |

### 7.4 仲裁管辖

- 由劳动合同履行地或用人单位所在地的劳动争议仲裁委员会管辖
- 双方当事人分别向两地申请仲裁的，由劳动合同履行地的仲裁委员会管辖

## 八、社区工作者特殊考点

### 8.1 社区工作者的法律身份

**核心要点**：社区工作者与街道（乡镇）、社区之间劳动关系的认定

- **签订劳动合同**：社区工作者应当与用人单位签订劳动合同，建立劳动关系
- **试用期内考核**：社区工作者试用期考核不合格的处理
- **岗位调整**：合同期内调整工作岗位的条件和程序

### 8.2 社区工作者管理相关规定

**常见规定**：
- 社区工作者实行合同管理
- 薪酬待遇按照相关规定执行，建立正常增长机制
- 依法参加社会保险，缴存住房公积金
- 实行考勤制度，遵守工作纪律

### 8.3 考试重点提示

社区工作者考试中劳动法部分需特别关注：

1. **劳动合同签订**：必考！注意无固定期限劳动合同的签订条件
2. **试用期规定**：同一单位同一员工只能约定一次试用期
3. **加班工资**：三种加班类型及计算基数
4. **年休假**：累计工作年限与休假天数对应关系
5. **工伤认定**：7种应当认定+3种视同工伤的情形
6. **劳动争议程序**：一裁两审制，仲裁前置

### 9.1 数字类考点汇总

| 16岁 | 最低就业年龄（文艺体育特种工艺除外） |
| 18岁 | 完全民事行为能力、禁止招用未成年工从事禁忌劳动 |

| 比例 | 关联内容 |

## 十、典型题目演练

### 题目1（单选题）
用人单位与劳动者约定"工伤自负"的条款效力如何？

**答案**：无效。该条款违反了法律的强制性规定，属于用人单位免除自己法定责任、排除劳动者权利的情形，劳动合同无效。

### 题目2（多选题）
劳动者可以依法解除劳动合同的情形包括？

**答案**：ABCD
- A. 与用人单位协商一致
- B. 提前30日以书面形式通知用人单位
- C. 在试用期内提前3日通知
- D. 用人单位未依法缴纳社会保险

### 题目3（判断题）
劳动者在上下班途中因本人主要责任的交通事故受伤，应当认定为工伤。

**答案**：错误。必须是"非本人主要责任"的交通事故才能认定工伤。

### 题目4（简答题）
简述劳动合同的法定必备条款。

**参考答案**：
1. 用人单位的名称、住所和法定代表人或主要负责人
2. 劳动者的姓名、住址和居民身份证号码
3. 劳动合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护和劳动条件

## 结语

劳动法是社区工作人员招聘考试的必考内容，知识点多且细。建议备考时：

1. **构建框架**：先建立知识体系框架，再填充具体内容
2. **抓住重点**：重点掌握高频考点（试用期、加班工资、工伤认定等）
3. **对比记忆**：善用表格对比、数字记忆等方法
4. **多做练习**：通过真题演练巩固知识点

祝您考试顺利！
